Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text (italic) <em>italic text</em> Italic text
strong strong make strong text (bold) <strong>Bold text</strong> Bold text
a anchor make hyperlink <a href = "https://www.google.com/">Google</a> Google
img image include an image <img src="images/kitten.jpg" alt="Kitten" height="40" width="40">
